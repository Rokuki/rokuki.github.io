S_green : (1461 - 454) * (2300 - 300) = 2014000

sum_area = 2001996 + 2014000 = 4015996

重叠部分：

交叉部分面积：

S_overlapping = （1392 - 454） * （2300 - 367）= 1813154

计算IOU：
iou = (S_overlapping / sum_area - S_overlapping) * 1.0
= (1813154 / 4015996 - 1813154) * 1.0
= 0.82309
