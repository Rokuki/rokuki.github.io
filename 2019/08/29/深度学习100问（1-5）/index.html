深度学习100问（1-5）

2019-08-29

1. 什么是交叉验证？

将原始数据集分为K（通常K大于3）组不同的子集，每次选K-1组子集进行训练，剩下一组子集作为验证集，如此进行K次训练并得到K个模型。这K个模型在各自验证集上的错误率的平均作为对分类器的评价。

2. 如何选择训练样本个数？

PAC（Probably Approximate Correct）学习理论。

3. 衡量模型的评价指标有哪些？

  1. 正确率：正确率是我们最常见的评价指标，accuracy = （TP+TN）/(P+N)，这个很容易理解，就是被分对的样本数除以所有的样本数，通常来说，正确率越高，分类器越好；
  2. 精度：精度是精确性的度量，表示被分为正例的示例中实际为正例的比例，precision=TP/（TP+FP）
  3. 召回率：召回率是覆盖面的度量，度量有多个正例被分为正例，recall=TP/(TP+FN)=TP/P=sensitive，可以看到召回率与灵敏度是一样的。
    对于类别C来说，模型在测试集上的结果可以分为：
  • 真正例(TP，True Positive)，一个样本的真实类别为C并被模型正确预测为C。
  • 假负例(FP，False Negative)，一个样本的真实类别为C并被模型错误预测为其他类。
  • 假正例(FP, False Positive), 一个样本的真实类别为其他类，模型预测为C。
  • 真负例(TN, True Negative), 一个样本的真实类别为其他类并被模型预测为其他类。

4. 混淆矩阵

真实情况 预测结果
Positive Negative
正例 TP (真正例) FN (假负例)
负例 FP (假正例) TN (真负例)

5.什么是提前终止（Early Stopping）

在训练的的每次迭代时，将得到的模型在验证集上进行测试，并计算错误率，当在验证集上的错误率不再下降，就停止迭代。作用是防止过拟合。
