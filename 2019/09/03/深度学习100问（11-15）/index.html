深度学习100问（11-15）

Sep 03 2019

11.深度学习解决什么问题？

深度学习解决的关键问题是贡献度分配问题（Credit Assignment Problem, CAP）,即一个系统中不同组件或其参数对最终系统输出结果的贡献或影响。

12. 神经网络训练和预测的基本流程是？

训练：输入数据 -> 网络层前向传播 -> 计算损失 -> 网络层反向传播梯度 -> 更新参数
预测：输入数据 -> 网络层前向传播 -> 输出结果

13. 什么是梯度？

梯度的本意是一个向量（矢量），表示某一函数在该点处的方向导数沿着该方向取得最大值，即函数在该点处沿着该方向（此梯度的方向）变化最快，变化率最大（为该梯度的模）
